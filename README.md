# Lógica de Programação em Python

Este repositório contém exercícios e atividades acadêmicas desenvolvidas durante os estudos em programação, com foco em lógica de programação e fundamentos da linguagem Python.

Os códigos foram criados como parte de atividades de faculdade e exercícios práticos para fixação de conceitos básicos.

## Conteúdos praticados
- Estruturas de repetição (for e while)
- Estruturas condicionais (if/else)
- Vetores (listas)
- Matrizes
- Validação de dados de entrada
- Cálculo de média, soma, maior e menor valor
- Ordenação simples de valores

## Arquivos do repositório
- `entrevista.py` – Exercício com coleta de dados, validação de entrada e cálculos estatísticos.
- `maior_menor_vetor.py` – Identificação do maior e menor valor em um vetor.
- `vetor.py` – Operações básicas com vetores.
- `matriz.py` – Manipulação de matriz e identificação do maior valor.
- `notas.py` – Validação de notas, cálculo de média e ordenação.

## Objetivo
Praticar lógica de programação e desenvolver raciocínio lógico aplicado à resolução de problemas computacionais.
